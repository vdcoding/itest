itest是用python3开发的http接口自动化测试工具，基于excel文件格式的测试数据来驱动测试的执行，并生成html格式的详细的测试报告。该工具是本人结合实际情况开发的一款简单的自动化测试工具，个人觉得其有以下几点优势：

1、基于命令行方式执行，便于CI测试及扩展

2、基于excel文件维护测试用例集，相比于jmeter、postman等GUI界面的测试工具，在维护测试用例上更加方便
